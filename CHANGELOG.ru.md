## [1.4.1] - 2026-02-25

### 🔧 Исправлено
- **Устранены предупреждения компилятора**: билд теперь полностью чистый
- **`@param:ApplicationContext`**: исправлена цель аннотации Hilt в `SettingsRepository` для совместимости с Kotlin 2.2.0+ (KT-73255)
- **Устаревший `hiltViewModel()`**: выполнена миграция с артефакта `hilt-navigation-compose` на `hilt-lifecycle-viewmodel-compose`
- **Хардкод строк**: подписи к иконке показа/скрытия пароля (`show_password`, `hide_password`) вынесены в `strings.xml` / `strings-ru.xml`
- **Правила бэкапа**: файл настроек DataStore теперь явно исключён из автоматического бэкапа Android и переноса между устройствами во избежание утечки сохранённого пароля

### 📦 Технические детали
- Обновлён `versionCode` до 7
- Обновлён `versionName` до "1.4.1"
- Заменена зависимость: `hilt-navigation-compose` → `hilt-lifecycle-viewmodel-compose:1.3.0`

## [1.4.0] - 2025-12-19

### 🌍 Добавлено - Локализация
- **Английская локализация** (по умолчанию для всех языков)
- **Русская локализация** (для русского системного языка)
- Автоматическое определение языка на основе системных настроек
- Английский используется как резервный для всех не-русских языков

### 🎨 Добавлено - Улучшения UI/UX
- **Адаптивный макет**: Автоматическое переключение между 1-колоночной (большие экраны ≥700dp) и 2-колоночной (маленькие экраны <700dp) сеткой чекбоксов
- **Всплывающие подсказки**: Полезные подсказки теперь появляются поверх контента, не нарушая макет
- **Компактный дизайн**: Оптимизированы отступы и поля для лучшего использования экрана
- **Переименована метка**: "Исключать повторяющиеся" → "Исключать повторы" (влезает в одну строку)

### 🏗️ Изменено - Рефакторинг архитектуры
- **Чистая архитектура**: Полная реструктуризация на слои Domain/Data/UI
- **Компонентный UI**: Созданы переиспользуемые компоненты:
    - `CheckboxRow.kt` - Унифицированный чекбокс с всплывающей подсказкой
    - `PasswordField.kt` - Поле ввода пароля с переключением видимости
    - `LengthSlider.kt` - Компактный слайдер для длины пароля
    - `StrengthIndicator.kt` - Визуальный индикатор надёжности с анимациями
- **Миграция на StateFlow**: Заменили LiveData на StateFlow для управления состоянием
- **Улучшения ViewModel**: Лучшее разделение ответственности с помощью extension-функций
- **Интеграция Hilt DI**: Правильное внедрение зависимостей во всём приложении

### 🛠️ Изменено - Технические улучшения
- **Утилита HapticFeedback**: Вынесена логика тактильной отдачи в отдельный утилитарный класс
- **Enum PasswordStrength**: Создан enum с companion object для расчётов надёжности
- **Extension-функции**: Добавлены функции маппинга между слоями для более чистого кода
- **Минимизирована рекомпозиция**: Оптимизирована производительность Compose
- **Обновлены зависимости**: Kotlin 2.2.0, AGP 8.13.2, Compose BOM 2025.12.01

### 🔧 Исправлено
- **API буфера обмена**: Обновлено использование стандартного Android ClipboardManager вместо устаревшего Compose Clipboard
- **Переполнение текста**: Исправлено обрезание текста в метках чекбоксов
- **Очистка импортов**: Удалены неиспользуемые импорты из всех файлов
- **Предупреждение AndroidManifest**: Удалён устаревший атрибут package
- **Предупреждения аннотаций**: Исправлены цели аннотаций Hilt для совместимости с Kotlin 2.2.0

### 📦 Технические детали
- Обновлён `versionCode` до 6
- Обновлён `versionName` до "1.4.0"
- Target SDK: 36
- Kotlin: 2.2.0
- AGP: 8.13.2
- Compose BOM: 2025.12.01
- Hilt: 2.57.2

## [1.2.0] - 2024-XX-XX

### Добавлено
- Индикатор надёжности пароля с цветовой визуальной обратной связью
- Расчёт надёжности в реальном времени на основе энтропии
- Опция исключения похожих символов (i I 1 l o O 0)
- Опция исключения повторяющихся символов
- Тактильная отдача при нажатии кнопок и копировании
- Реализация Material 3 дизайна
- DataStore Preferences для сохранения настроек

### Изменено
- Улучшен макет UI и отступы
- Расширено поле пароля с переключением показа/скрытия
- Обновление до Jetpack Compose
- Миграция на компоненты Material 3

### Технические изменения
- Реализована архитектура MVVM
- Добавлена ViewModel с LiveData
- Интегрирован Hilt для внедрения зависимостей
- Добавлен DataStore для локального хранилища
