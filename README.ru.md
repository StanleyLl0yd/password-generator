## ✨ Возможности

### 🔐 Генерация паролей
- **Гибкие наборы символов**: строчные, заглавные буквы, цифры, спецсимволы
- **Настраиваемая длина**: от 4 до 64 символов
- **Умные опции**:
    - Исключение повторяющихся символов
    - Исключение похожих символов (i I 1 l o O 0) для лучшей читаемости
- **Индикатор надёжности в реальном времени**: визуальная обратная связь с цветным прогресс-баром
- **Копирование в один клик** с тактильной отдачей

### 🎨 Современный UI/UX
- **Адаптивный макет**: автоматическое переключение между 1 или 2 колонками в зависимости от высоты экрана
- **Всплывающие подсказки**: полезные подсказки, которые появляются поверх контента, не нарушая макет
- **Material 3 Design**: следует последним рекомендациям Google по дизайну
- **Плавные анимации**: отшлифованные переходы и микро-взаимодействия
- **Тёмная/Светлая тема**: автоматически на основе системных настроек

### 🌍 Локализация
- **English** (по умолчанию для всех языков)
- **Русский**
- Автоматическое определение языка на основе системных настроек

### 🔒 Приватность и Безопасность
- **100% офлайн**: не требуется доступ к сети
- **Без сбора данных**: никакой аналитики, отслеживания или рекламы
- **Только локальное хранилище**: пароли генерируются в памяти, опционально сохраняются локально
- **Защита бэкапа**: сохранённые настройки явно исключены из автоматического бэкапа Android
- **Открытый исходный код**: полная прозрачность кода

### 🏗️ Техническое совершенство
- **Чистая архитектура**: разделение Domain/Data/UI
- **Паттерн MVVM**: управление состоянием на основе StateFlow
- **Hilt DI**: внедрение зависимостей для тестируемости
- **Jetpack Compose**: современный декларативный UI
- **Компонентный подход**: переиспользуемые UI-компоненты
- **Оптимизированная производительность**: минимальная рекомпозиция

## 📦 Установка

### Из исходного кода

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/StanleyLl0yd/password-generator.git
   cd password-generator
   ```

2. **Откройте в Android Studio**:
    - Android Studio Hedgehog (2023.1.1) или новее
    - JDK 11 или выше

3. **Синхронизируйте Gradle**:
    - Позвольте Android Studio синхронизировать зависимости

4. **Соберите и запустите**:
   ```bash
   ./gradlew assembleDebug
   ./gradlew installDebug
   ```

### APK релиз

Скачайте последний APK из [Releases](https://github.com/StanleyLl0yd/password-generator/releases)

## 📁 Структура проекта

```
app/src/main/java/com/sl/passwordgenerator/
├── data/
│   └── SettingsRepository.kt          # Хранение в DataStore
├── domain/
│   ├── model/
│   │   ├── GeneratorPreferences.kt    # Модель настроек пользователя
│   │   ├── PasswordGenerationConfig.kt
│   │   ├── PasswordGenerationResult.kt
│   │   └── PasswordStrength.kt        # Enum надёжности с логикой
│   ├── usecase/
│   │   └── PasswordGenerator.kt       # Основная логика генерации
│   └── PasswordConstants.kt           # Наборы символов и константы
├── ui/
│   ├── components/
│   │   ├── CheckboxRow.kt             # Переиспользуемый чекбокс с подсказкой
│   │   ├── LengthSlider.kt            # Слайдер длины пароля
│   │   ├── PasswordField.kt           # Поле ввода с переключением видимости
│   │   └── StrengthIndicator.kt       # Визуальный индикатор надёжности
│   ├── theme/
│   │   ├── Color.kt                   # Цветовая палитра Material 3
│   │   ├── Theme.kt                   # Конфигурация темы
│   │   └── Type.kt                    # Типографическая система
│   ├── PasswordGeneratorScreen.kt     # Главный composable экран
│   ├── PasswordGeneratorViewModel.kt  # Состояние и бизнес-логика
│   └── PasswordGeneratorUiState.kt    # Data class состояния UI
├── util/
│   └── HapticFeedback.kt              # Утилита вибрации
├── MainActivity.kt                     # Точка входа
└── PasswordGeneratorApplication.kt    # Класс приложения Hilt
```
